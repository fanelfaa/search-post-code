# Search Indonesian Post Code
Welcome to this simple project, guys

## Project Stack
```
React
Typescript
Chakra UI
react-select
React Router
Alova (alternative to axios, but much simpler)
```

## How to install
run this command on terminal
```
pnpm install
```

## How to start on local
```
pnpm run dev
```

> Note other command, see in `package.json`
